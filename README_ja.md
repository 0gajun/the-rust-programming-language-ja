[TRPL](https://doc.rust-lang.org/book/)を主としたRustドキュメントの翻訳プロジェクトです。

# ビルド方法

`rustbook`コマンドが必要です。rustコンパイラをソースからビルドした時にstage2にいる
ようです。汎用のインストール方法は知りません。

```
make RUSTBOOK=/path/to/rustbook
```
